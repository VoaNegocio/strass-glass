# Strass Glass - Guia de Branding

## Paleta de Cores

As cores foram selecionadas para transmitir confiança técnica, solidez e precisão.

### Cores Principais

| Nome do Token | Valor Hex | Descrição |
| :--- | :--- | :--- |
| **`strass-blue`** | `#0F172A` | **Slate 900**. Cor primária. Usada em fundos, textos principais e elementos de "Engenharia/Técnica". Transmite profundidade e seriedade. |
| **`strass-accent`** | `#0EA5E9` | **Sky 500**. Cor de destaque. Usada para "vidro", "clareza" e pontos de atenção positivos. |

### Cores Funcionais

| Nome do Token | Valor Hex | Descrição |
| :--- | :--- | :--- |
| **`strass-alert`** | `#EF4444` | **Red 500**. Usada estrategicamente na seção de "Problemas" (atrasos, prejuízos) para gerar alerta visual. |
| **`strass-gray`** | `#F8FAFC` | **Slate 50**. Fundo geral da página. Garante leitura limpa e contraste suave. |
| **`strass-border`** | `#E2E8F0` | **Slate 200**. Bordas sutis para separar conteúdo sem pesar o layout. |

---

## Tipografia

A tipografia foi escolhida para maximizar a legibilidade e a estética técnica.

- **Fonte Principal:** `Inter` (Google Fonts)
- **Características:** Sans-serif, moderna, altamente legível em UI, com excelente suporte a pesos variados (Regular, Medium, Bold).

---

## Logotipo

O logotipo deve ser utilizado preferencialmente em sua versão horizontal para manter a integridade da marca no cabeçalho.

- **Arquivo:** `/logo/logo-strasglass-horizontal-curvas.png`
- **Uso:** Fundo escuro (negativo/branco) ou Fundo claro (original/azul).

---

*Nota: As cores foram extraídas da configuração do sistema (`tailwind.config.js`) para garantir fidelidade ao que está implementado.*
